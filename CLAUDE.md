# RHT-NAV Project Rules

## DATA INTEGRITY RULE

**PERMANENT — applies to every pass, every contributor, every model.**

NEVER generate, simulate, fabricate, or estimate data. Every number, statistic, or geographic data point must come from an explicit, credible government source: CMS, US Census Bureau, FCC Broadband Data Collection, HRSA AHRF, HRSA HPSA, CDC PLACES, ArcGIS Hub, state health departments, Cecil G. Sheps Center, KFF, or equivalent peer-reviewed/government sources.

If real data for a given module or state is not available in the current pass, display a clearly visible placeholder:

> "Data coming soon — [source name and what field it will populate]"

Do not invent numbers to fill space. Do not round-trip estimates. Do not use LLM-generated statistics. Every displayed number must be traceable to a cited primary source.

This rule is non-negotiable and overrides any instruction to "fill in" or "estimate" data.
